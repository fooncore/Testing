# Foonex Solutions — Corporate Website

## 1. Project Description
A modern, professional corporate website for Foonex Solutions, an IT consulting company based in Edmonton, Canada. The site targets small and medium-sized businesses (clinics, professional firms, growing organizations) who need cybersecurity and IT infrastructure consulting. Core value: trust, innovation, and clarity delivered through 20+ years of enterprise IT experience.

## 2. Page Structure
- `/` — Home (hero, about, services, industries, why us, CTA, footer)

## 3. Core Features
- [ ] Hero section with headline and consultation CTA
- [ ] About Us section with founder profile and mission/vision
- [ ] Core Services section (6 services)
- [ ] Industries We Serve section (4 industries)
- [ ] Why Choose Us trust-signal section
- [ ] Call-to-Action consultation booking section with form
- [ ] Responsive navigation with smooth scroll anchors
- [ ] Professional footer with links and contact info

## 4. Data Model Design
No database required for initial phase. Static corporate website with form submissions handled via Readdy form endpoint.

## 5. Backend / Third-party Integration Plan
- Supabase: Not needed for Phase 1
- Shopify: Not needed
- Stripe: Not needed
- Forms: Readdy built-in form submission for consultation booking

## 6. Development Phase Plan

### Phase 1: Homepage Design & Development
- Goal: Build a complete, polished single-page corporate homepage with all sections
- Deliverable: Fully responsive homepage with all 7 content sections, smooth animations, and functional consultation form

### Phase 2: Enhancements & Polish (Future)
- Goal: Add additional pages (e.g., dedicated Services page, Contact page) if needed
- Deliverable: Extended site with multi-page routing
